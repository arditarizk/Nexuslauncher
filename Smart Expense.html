Setup Cloud Simpel ☁️

  1. Buka Google Sheets, buat file baru.
  2. Tulis ini di baris paling atas (Header):
    Tanggal | Nama | Kategori | Nominal | Catatan | ID | Waktu Input
  3. Klik menu Extensions > Apps Script.
  4. Hapus semua isinya, lalu klik dan Paste.
  5. Klik Deploy > New Deployment > tipe Web App.
  6. Akses pilih Anyone, klik Deploy, copy URL-nya ke kotak di atas! ✨
